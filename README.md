# 既読判定URL発行・閲覧アプリ



## ユビキタス言語
| 単語        | 説明                                              | 
| ----------- | ------------------------------------------------- | 
| アクセス判定URL | 企業名とIDが対応づけられた既読判定を行うためのURL | 
| アクセス        | アクセス判定URLを開くこと                                     | 
| アクセス確認 | アクセス判定URLに紐づくアクセス履歴を閲覧すること |
